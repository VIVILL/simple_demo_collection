# simple_demo_collection
一些简单的demo合集。以下为简单说明。

1.AndroidTcpClient：用于控制网络继电器设备。运行环境：Android5.

2.WebView:WebView的简单使用。运行环境：Android7.

3.FullScreenTest:界面全屏   运行环境：Android7.

4.EthTest: android获取以太网ip并执行ping ip地址命令的简单demo.
